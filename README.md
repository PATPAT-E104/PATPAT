# 🐾 Intro

✅ 유기견 보호소와 유기견, 안락사에 대해 무심한 사람들을 대상으로 **_관심을 유발_** 한다.

✅ 전국 유기견 보호소 사이트의 **_통합적이고 효율적인 관리_** 를 가능하게 한다.

✅ **실시간 알림** 서비스

✅ 개인 - 보호소간 **_비대면 화상상담_**

✅ 손쉬운 **_봉사 신청 및 관리_**
